# Hunting Coder
A Blog website where user can read blogs, add blog after creating account


# Tech used 

**Frontend/Backend :** Next Js, Tailwind CSS

**Database :** MongoDB

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn install
```

Start the program

```bash
  yarn dev
```
